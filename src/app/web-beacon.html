Web Beacon Trackers

A "Web Beacon" is a small, usually transparent 1x1 image - it can be used to silently track when an email is opened.

How does it work?

Most email clients use HTML for message formatting. HTML allows images to be embedded in the message. When you open an HTML format email, your email client makes a request for the image.

There are thus 3 components to a Web Beacon tracker:

  1. Send an HTML email with a specially coded img tag with the web beacon URL.
  2. A web server that listens for requests for the web beacon image and records information about each request, which may include the requester's email address, their IP address, the time of the request, which email client they use, etc.
  3. A database where information about each person to whom an email was sent is linked with the tracking information obtained when their email client makes a request for the web beacon embedded within.

Protecting your privacy from Web Beacons

Since web beacons work by tracking a request for a remote resource (e.g., a small image file) you can completely defeat them... IF you can prevent your mail tool from requesting the beacon!!
